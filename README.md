About the project


Project: Sanity

Set in 2087 New York, you take the role of an unnamed Private investigator, drawn into a case in the central district of the city because of a string of unexplained disappearances tied to one hotel. Beneath the polished exterior of a once-thriving hotel hides something far more sinister: a cult lurking in the shadows. Across three escalating levels, you will descend deeper into the unknown, confronting its followers and uncovering a truth never meant to be surfaced and the ultimate goal of the Cult.



About the repo 

This GitHub repository will serve as the central hub for all developers working on the project, providing a single, organized space to access essential information and stay aligned as a team. It will be used to share progress updates, track changes, and communicate ongoing work so that everyone remains informed and coordinated throughout development.

In addition, the repository will house all game-related documentation and resources, including design documents, technical specifications, asset references, and implementation guidelines. By centralizing these materials, the repo will help streamline workflows, reduce miscommunication, and ensure that all team members have the tools and context they need to contribute effectively to the project.
